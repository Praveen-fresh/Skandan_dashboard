Load Your Google Sheet

Paste the base publish URL and enter the Sheet ID (gid) for each tab.

How to find each tab's gid:
1. Open your Google Sheet → click a tab → File → Share → Publish to web
2. Select that sheet tab → CSV → Publish → copy the link
3. The link will look like: …/pub?gid=1234567890&output=csv
4. Copy the number after gid= — that's the gid for that tab.

Or: right-click the tab → "Copy link" in the browser address bar — the URL ends with #gid=XXXXXXXX
